# e-POM (Electronic Private Office Management)

The e-POM application is a digital platform designed to help leadership offices efficiently manage Time, Information, and Action. It transitions manual and fragmented processes into a centralized, structured, and trackable digital workflow, improving decision-making and operational efficiency.

## Overview
This application follows a 3-tier architecture:
- **Frontend**: React (Vite, TypeScript, Tailwind CSS)
- **Backend**: Flask (Python)
- **Database**: SQLite
